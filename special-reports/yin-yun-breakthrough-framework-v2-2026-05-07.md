## 1. 음운(陰雲) / 양운(陽雲) 정의 — Ichimoku Cloud 기반

### 음운 (Bearish Cloud, 적색 구름)
- **수식**: Senkou Span A < Senkou Span B
- **의미**: 약세 추세 구간 / 위에서 누르는 저항대
- **가격 위치 해석**:
  - 가격이 음운 안 → 약세 지속, 횡보 가능
  - 가격이 음운 아래 → 명확한 약세
  - 가격이 음운 위 → **돌파 임박** (강세 전환 진입 신호)

### 양운 (Bullish Cloud, 녹색 구름)
- **수식**: Senkou Span A > Senkou Span B
- **의미**: 강세 추세 구간 / 아래에서 받쳐주는 지지대
- **가격 위치 해석**:
  - 가격이 양운 위 → **강세 지속, HOLD/추매**
  - 가격이 양운 안 → 강세 약화, 추가 확인 필요
  - 가격이 양운 아래 → **양운하락** (약세 전환 신호)

### Ichimoku 핵심 5선
| 선 | 한글명 | 계산 (일봉 기준) | 활용 |
|----|------|----------------|-----|
| Tenkan | 전환선 | (9일 최고 + 9일 최저) / 2 | 단기 모멘텀 |
| Kijun | 기준선 | (26일 최고 + 26일 최저) / 2 | 중기 추세 |
| Senkou A | 선행스팬 1 | (Tenkan + Kijun) / 2 (26일 선행) | 구름 상단 |
| Senkou B | 선행스팬 2 | (52일 최고 + 52일 최저) / 2 (26일 선행) | 구름 하단 |
| Chikou | 후행스팬 | 종가 (26일 후행) | 반대편 추세 확인 |

## 2. 4가지 핵심 시나리오 + LONG/SHORT 판단

| # | 시나리오 | 가격 위치 변화 | 신호 | LONG/SHORT |
|---|---------|--------------|------|-----------|
| **A** | **음운돌파 성공** | 음운 안/아래 → 음운 위 + 2일 종가 유지 | 🟢🟢 강한 강세 전환 | **LONG 진입** |
| **B** | **양운돌파 성공 (양운하락)** | 양운 위 → 양운 아래 + 2일 종가 유지 | 🔴🔴 강한 약세 전환 | **SHORT 진입** (or LONG 정리) |
| **C** | **음운돌파 실패** | 음운 위 시도 → 다시 음운 안 침강 | ⚠️ Fake Breakout | **관망** (재돌파 대기) |
| **D** | **양운유지** | 가격이 양운 위 지속 | ✅ 강세 지속 | **HOLD / 분할 매집** |

### Daniel 핵심 질문 응답 — "돌파 시도 후 재돌파 시 LONG/SHORT 판단"

**1차 음운돌파 실패 (시나리오 C)**:
- 종가 기준 음운 상단 위 1일 마감 후, 다음 날 다시 음운 안으로 진입
- 거래량이 평균 미만이면 명백한 **Fake Breakout** → 관망
- 거래량이 평균 이상이었으나 다음날 침강 → 매물 부담 → 추가 검증

**재돌파 신호 (1차보다 더 강한 신호 — 정량 검증)**:
| 검증 항목 | 1차 실패 후 재돌파 진입 조건 |
|---------|----------------------------|
| **1. 지지 확인** | 1차 실패 후 SMA50 또는 SMA200에서 **거래량 동반 양봉 지지** (최소 3-5일 횡보 후 반등) |
| **2. 거래량 증가** | 2차 돌파 시 거래량이 1차 시도 시점 대비 **+30% 이상** 증가 |
| **3. 구름 두께 축소** | Senkou A vs B 갭이 1차 시도 때보다 **20% 이상 축소** (음운 약화) |
| **4. RSI Divergence** | 가격은 1차 실패 저점 ≥ 2차 시도 저점이지만 **RSI는 2차가 더 높음** (Bullish Divergence) |
| **5. Tenkan/Kijun 골든** | 전환선 > 기준선 (단기 모멘텀 확정) |

→ **5개 항목 중 4개 이상 충족 시 재돌파 = 1차보다 더 강한 LONG 신호**  
→ **3개 이하 충족 시 = 또 다른 Fake 가능성, 비중 50% 이하로 진입**

## 3. 매집 / 진입 / 익절 — 정량 Timing Rule

### 🟢 LONG 음운돌파 시나리오 (3-Stage 분할)

#### Stage 1 — 매집 (Pre-Entry, 사전 진입)
**진입 시점**: 음운 상단 -3% ~ -5% 접근 시 (돌파 임박 구간)
| 조건 | 임계값 |
|------|-------|
| 가격 vs 음운 상단 | -3% ~ -5% 이내 접근 |
| RSI | 45-55 (중립) |
| 거래량 | 20일 평균의 1.0-1.3x |
| SMA200 | 가격이 SMA200 위 (장기 추세 유지) |
| 비중 | 목표의 **25-50%** |
| 손절선 | SMA200 -7% |

**왜 매집?** — 돌파 직전 진입 시 단가 우위 + 돌파 실패 시 손실 제한

#### Stage 2 — 확정 진입 (Confirmed Entry)
**진입 시점**: 음운돌파 확정 후 (종가 2일 유지)
| 조건 | 임계값 |
|------|-------|
| 종가 위치 | 음운 상단 위 + **2일 연속 종가 유지** |
| 거래량 | 20일 평균의 **1.5x 이상** |
| RSI | 55-70 (강세 모멘텀, 과매수 70 이전) |
| Tenkan vs Kijun | 전환선 > 기준선 |
| 비중 추가 | **25-50% 추가** (총 75-100%) |
| 손절선 | 진입가 -7~10% 또는 SMA50 종가 이탈 |

**왜 확정 진입?** — Fake Breakout 위험 제거 + 가장 강한 모멘텀 구간

#### Stage 3 — 익절 (Take Profit, 분할)
**룰: 33% / 33% / 33% 분할 매도**

| 단계 | 매도 비중 | 트리거 (먼저 도달하는 것) |
|-----|---------|------------------------|
| **1차 익절** | 33% | RSI 75 도달 **또는** 진입가 +20% |
| **2차 익절** | 33% | Bollinger 상단 +2 ATR **또는** 진입가 +35% |
| **3차 익절 (Trailing)** | 33% | SMA50 종가 이탈 (장기 추세 종료 시그널) |

**왜 분할?** — 추세 지속 가능성 보존 + 단계별 수익 확정

#### 🚨 즉시 이탈 (Stop Loss)
- 종가가 음운 상단을 **2일 이상 하회** → 음운하락 → 비중 50% 즉시 매도
- 거래량 급증한 음봉 + 종가 하단 → Fake Breakout 확정 → 비중 100% 매도
- SMA50 종가 이탈 → 단기 추세 종료 → Trailing 33% 청산

### 🔴 SHORT (양운하락) 시나리오 — 일반 retail 활용
- 일반 retail은 short 포지션 잘 안 하므로 → **LONG 보유 종목 정리/축소 신호로 활용**
- 양운 하단 -3% 접근 + RSI 50 미만 → 보유 비중 33% 축소
- 양운 하단 종가 2일 하회 → 추가 33% 축소 (총 66%)
- SMA200 이탈 → 100% 청산

## 4. 보조 지표 — 정확도 향상

### 4-1. RSI 활용 (Wilder's 14-day)
| RSI 구간 | 의미 | 액션 |
|---------|------|-----|
| 0-30 | 과매도 | 매집 신호 (단, 추세 확인 필수) |
| 30-45 | 약세 | 추가 하락 가능, 관망 |
| 45-55 | 중립 | **음운돌파 매집 구간** ⭐ |
| 55-70 | 강세 | **음운돌파 확정 진입** ⭐ |
| 70-80 | 과매수 초입 | 1차 익절 33% |
| 80+ | 극단 과매수 | 2차 익절 33% |

### 4-2. Bollinger Band (20-day, 2σ)
- 상단 돌파 + 종가 2일 유지 → 강세 모멘텀 확인 (음운돌파 보조 시그널)
- 상단 +2 ATR 도달 → 2차 익절 트리거
- 중앙선 (SMA20) 종가 이탈 → 단기 모멘텀 종료

### 4-3. ATR (Average True Range, 14-day)
- 손절선 폭 결정: **진입가 - 1.5 ATR**
- 익절 트리거: **진입가 + 3 ATR (1차) / +5 ATR (2차)**
- ATR 확장 = 변동성 증가 = 비중 축소

### 4-4. Volume Profile
- 음운돌파 시 거래량 ≥ 20일 평균 1.5x → 진정 돌파
- 거래량 < 평균 → Fake Breakout 의심
- POC (Point of Control) — 거래량 최대 가격대 → 핵심 지지/저항
